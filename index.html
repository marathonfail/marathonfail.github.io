Rules

  • This is a classic word building game. Your word should start with the ending character of the previous word in the chain.
  • Use ONLY lowercase English alphabets, NO symbols, punctuations or spaces.
  • The price is 0.01 ETH for adding a word
  • While there is no check on the kind of word you add, please add meaningful/thoughtful words. We will provide an option for others to tip for meaningful words.
  • 1% of the tip will be withheld by the contract.
  • Help create the longest word building game ever!
  • Install MetaMask chrome extension as this app depends on MetaMask for sending transactions.
